# 👩‍💼 Employee Manager – Full Stack App (Angular + GraphQL + MongoDB)

A full-stack Employee Management system built using Angular frontend, GraphQL API, Node.js backend, and MongoDB. This app allows registered users to manage employees efficiently with CRUD operations and profile uploads.

---

## 🚀 Features

- ✅ User Authentication (Register / Login)
- ✅ JWT-based session management
- ✅ Angular Route Guards to protect routes
- ✅ Employee CRUD (Create, Read, Update, Delete)
- ✅ Profile picture upload using Multer
- ✅ Search employees by department and position
- ✅ Responsive UI (Bootstrap)
- ✅ Deployed on cloud platforms

---

## 🧱 Tech Stack

- **Frontend:** Angular, Apollo Angular, Bootstrap
- **Backend:** Node.js, Express.js, GraphQL
- **Database:** MongoDB Atlas
- **Authentication:** bcrypt, JWT
- **Deployment:** Vercel (Frontend), Render (Backend)
